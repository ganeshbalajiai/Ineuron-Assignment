1. To what does a relative path refer?
Answer: It refers to the current working directory.

2. What does an absolute path start with your operating system?
Answer: Absolute path starts with the root folder.

3. What do the functions os.getcwd() and os.chdir() do?
Answer: os.getcwd() it gives the current working directory, os.chdir() it changes the working directory.

4. What are the . and .. folders?
Answer: . is the current folder, .. is the parent folder.

Answer: c:/bacon\egg is the dir name, spam.txt is the base name
